# Nero's local neural voice (optional) — Increment 1: English via Kokoro.
#
# Install with:   pip install -r requirements-voice.txt
#
# Uses Kokoro through ONNX Runtime — fast, light, **no PyTorch**, and works on
# modern Python (incl. 3.13). The base app runs fine WITHOUT this; Nero just
# uses your browser's built-in voice until it's installed.
#
# On first use, the Kokoro model (~310 MB) downloads once from the kokoro-onnx
# release into your `models/` folder. A GPU is optional — it's fast on CPU.
#
# If synthesis ever fails on phonemization (rare), install espeak-ng once:
#   https://github.com/espeak-ng/espeak-ng/releases
